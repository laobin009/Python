For Package Pip and Package distribute, using file: get-pip.py to download and intall by runing it

Like this on WindowPowershell: PS C:\Python27> python get-pip.py

We will get pip, setuptool and Wheel

Setuptool is the better version of distribute

Then we pushd the directory to Scripts

For Package nose
PS C:\Python27\Scripts> .\pip install nose
For Package virtualenv
PS C:\Python27\Scripts> .\pip install virtualenv
